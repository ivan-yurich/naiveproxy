## рҹӨ” What is this

**NaiveProxy Manager** вҖ” a single bash script that turns a bare VPS into a fully protected proxy server with ad blocking and Telegram management.

## вҡ пёҸ Caddyfile вҖ” critical

```
{
    order forward_proxy before file_server
    servers :443 {
        protocols h1 h2 h3
    }
}

:443, your-domain.com {
  tls your@email.com
  forward_proxy {
    basic_auth USER PASS
    hide_ip
    hide_via
    probe_resistance
  }
  file_server { root /var/www/html }
}
```
